CVS (versão em desenvolvimento)

Para baixar a versão em desenvolvimento (recomendada para programadores apenas), você pode acessar nosso repositório CVS:

$ cvs -z3 -d :pserver:anonymous@abrigo.dyndns.org:/sniffdet co [module]

Os módulos disponíveis são:
  • sndet
  • web
  • docs
  • external

Você também pode utilizar a interface cvsweb, disponível em: http://abrigo.dyndns.org:81/cgi-bin/sniffdet-cvs/

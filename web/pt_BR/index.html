sniffdet - Detector Remoto de Sniffers

Descrição

O Sniffdet é uma implementação aberta de métodos de detecção remota de sniffers em redes TCP/IP. O projeto consiste em uma biblioteca com a implementação dos testes (libsniffdet) e uma ferramenta para sua execução.

O Sniffdet é distribuído sob a licença GPL (GNU General Public License)

Principais características

Biblioteca:
  • ICMP test
  • ARP test
  • DNS test
  • LATENCY test
  • Funções de uso geral (ping, init, finish)
Aplicação:
  • Suporte a arquivo de configuração
  • Suporte a arquivo com "targets"
  • Suporte a plugin para geração de relatórios
  • Core executado como usuário normal ("drop root privileges")

Lista de e-mail - Contribuições e Dúvidas

Se você deseja participar do desenvolvimento do sniffdet ou tem alguma dúvida ou sugestão, você pode se cadastrar em nossa lista enviando uma mensagem em branco para sniffdet-devel-request@lists.sourceforge.net com subscribe no campo assunto (subject).

Ou se preferir, acesse a interface web em http://lists.sourceforge.net/lists/listinfo/sniffdet-devel.

O idioma oficial da lista é o inglês (embora não sejam "proibidos" posts em português) e a mesma está aberta para:
  • Patches
  • Dúvidas
  • Bug reports
  • Discussão geral/feedback

O histórico da lista pode ser acessado através do SourceForge: SourceForge Archives

A página de nosso projeto na interface do SourceForge está em http://sourceforge.net/projects/sniffdet.
Lá você encontra uma interface para entrada de bugs, histórico, etc.

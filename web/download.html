All tarballs and RPM packages are signed with gpg. The fingerprint is: C484 537D D4D3 5BF9 FA6F 3E34 59CE A74B 825F 1270

CVS (latest development version)

To download the latest source code (for developers only), you may access the CVS repository using:

$ cvs -z3 -d :pserver:anonymous@abrigo.dyndns.org:/sniffdet co [module]

The available modules are:

  • sndet
  • web
  • docs
  • external

Or you can access the cvsweb interface, available at: http://abrigo.dyndns.org:81/cgi-bin/sniffdet-cvs/
